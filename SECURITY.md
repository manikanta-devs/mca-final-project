# Security Policy

## Supported Versions

Only the latest version of the AI Interview prep system receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

If you discover any security vulnerability in this capstone codebase, please report it via the following steps:

1. Draft a detailed explanation of the issue, including steps to reproduce.
2. Email the report privately to the project administrator or lead maintainer.
3. Do not open public issues on GitHub or public forums until a patch has been coordinated.

We will review your submission and aim to provide a response within 48 hours.
